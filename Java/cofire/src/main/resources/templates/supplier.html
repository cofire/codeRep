进入厂区注意事项

尊贵 的来宾：

非常欢迎您来到我们公司制剂药厂区！为了您的安全，在我们厂区检查/参观时，请注意以下情况：

仓库：

为高架仓库，我们给您准备了安全帽，请进入高架仓库时，在引导员的陪同下佩戴安全帽！

生产车间：

饰品、手机等私人物品不得带入车间，我们为您准备了收纳袋用于存放。进入制药生产车间时，请在引导员的陪同下进行更衣，我们为您准备了参观服 进入洁净区为您准备了洁净服，口罩及手套等物品。您的笔和笔记本最好也不要带入车间洁净区，我们在洁净区给您准备了笔和笔记本。如果您有 重度感冒、咳嗽、流鼻涕或体表有伤口的情况，请告知我们，我们公司规定此类人员是不得进入生产区域。

化验室：

介于安全考虑，进入化验室时，请佩戴护目镜。
